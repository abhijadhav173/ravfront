Summary of Changes

  • Remove: "Download" button on every document card (investor + admin views).
  • Remove: direct /storage/* file URLs as a public-ish delivery path.
  • Add: /investor/documents/[id] — new page-by-page PDF viewer.
  • Add: /admin/documents/analytics — engagement dashboard with page heatmap + per-investor breakdown.
  • Add: page-view tracking (session start, per-page duration, session end) — sent as batched events + final sendBeacon on tab close.
  • Add: inline email watermark in the viewer corner (low cost, high deterrent).
  • Keep: existing doc list structure, categories, admin upload flow — unchanged.
  • Keep: existing auth guards (investor.approved middleware, role:admin for analytics).
